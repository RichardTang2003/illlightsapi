illlights.API

概述

调用限制

由于服务器并发能力差，限制单ip每分钟60次调用，超过限制次数封锁600秒。

API

Saying

简介：这是一个我本人摘抄的句子数据库，API随机返回一条摘抄。

调用：get api.illlights.com/saying

暂时没有写路径和配置，待更新。

返回：JSON格式。例如[{"_id":1, "content":"我只能选择天空，决不跪在地上，以显出刽子手们的高大，好阻挡自由的风。", "type":"Poetry", "author":"北岛", "from":"宣告"}]，content为内容，type为来源文本类型，author作者，from为来源文本标题。部分数据可能缺少type和from项。
